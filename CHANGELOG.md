# 更新日志

本文件记录嘉嘉易学工具箱各版本的主要变更。

---

## v6.0 — API 系统

- **API v1 基础架构**: 新增 `/api/v1/` 路由层，统一请求校验、错误信封（`{ ok, data, error }`）
- **POST /api/v1/calendar**: 公农历转换、四柱干支、纳音五行、节气、称骨论命
- **POST /api/v1/bazi**: 八字排盘、大运排列、十维度分析、综合解读
- **POST /api/v1/meihua**: 梅花易数起卦（时间/数字/文字三种方式）与体用分析
- **POST /api/v1/liuyao**: 六爻纳甲排盘（手动/自动/数字/直接）与场景分析
- **POST /api/v1/qimen**: 奇门遁甲阴盘（王凤麟）+ 阳盘（拆补/置闰/茅山）排盘与分析
- **POST /api/v1/taiyi**: 太乙神数排盘（统宗/金镜，年/月/日/时四种计算类型）与分析
- **POST /api/v1/naming**: 姓名五格数理分析 + 智能起名推荐
- **设计原则**: 零修改现有页面代码，所有 API 为纯增量附加功能

## v5.1

- **玉匣通书**: "公农历查询"更名为"玉匣通书"，致敬《许真君玉匣记》
- **首页优化**: Hero 区域增加八卦装饰符号和朱砂分隔线，移除冗余的"开始查询"/"关于"按钮
- **太乙排盘**: 输入面板改为与奇门/六爻统一的 form-input 下拉框风格；经典案例展开时自动生成排盘图
- **穿衣指南**: 分隔线改为朱砂色渐变，与 section-title 装饰线风格一致
- **奇门排盘**: 阴盘九宫格添加 maxWidth 约束，与阳盘尺寸、字体完全对齐
- **梅花排盘**: 时间起卦的日期选择与"当前时间"按钮合并为一行
- **全站布局**: 梅花、六爻、奇门页面从 max-w-4xl 统一为 max-w-7xl 宽版
- **八字排盘**: 点击排盘后不再自动跳转到大运区域，保持原位让用户自行浏览
