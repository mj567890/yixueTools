# 易学工具平台 (YixueTools)

基于现代前端技术栈构建的中国传统易学在线工具集，提供八字排盘、万年历、称骨算命等功能，采用新中式视觉风格。

## 技术栈

- **框架**: Next.js 16 (App Router + Turbopack)
- **UI**: React 19 + TypeScript 5.9
- **样式**: Tailwind CSS 4
- **历法**: lunar-javascript 1.7.7

## 八字模块核心能力

- **排盘计算**: 四柱干支、十神、藏干、纳音、日主强弱、五行力量分析
- **大运系统**: 大运排列、起运年龄、五行评分、运势等级
- **流年系统**: 流年干支、合冲检测、评分、纳音、简洁断语
- **流年联动详解**: 原局联动(逐柱五行生克+冲合)、大运联动(五行互动+吉凶倾向)、四维断语(事业/财运/感情/健康)、关键提示(风险/机会)
- **十维度解读**: 性格/学业/事业/财运/婚姻/家庭/子女/健康/贵人/生命节律
- **五行雷达图**: SVG 五角雷达图，支持原局/+大运/+流年三层叠加对比

## 项目结构

```
src/
├── app/                    # Next.js App Router 页面
│   ├── page.tsx            # 首页（工具导航）
│   ├── bazi/page.tsx       # 八字排盘页
│   └── calendar/page.tsx   # 万年历页
├── components/
│   ├── bazi/               # 八字相关组件
│   │   ├── BaziPanel.tsx           # 四柱排盘面板
│   │   ├── AnalysisPanel.tsx       # 四维分析面板
│   │   ├── DaYunTimeline.tsx       # 大运时间线
│   │   ├── LiuNianGrid.tsx         # 流年网格(含点击交互)
│   │   ├── LiuNianDetailPanel.tsx  # 流年联动详情面板
│   │   ├── WuXingChart.tsx         # 五行雷达图
│   │   ├── DimensionCard.tsx       # 十维度解读卡片
│   │   ├── InterpretationPanel.tsx # 十维度解读面板
│   │   └── ElementAnalysis.tsx     # 五行分析
│   ├── calendar/           # 万年历组件
│   └── layout/             # 布局组件
└── lib/
    ├── lunar.ts            # 核心计算层(~1930行)
    └── bazi-interpretation.ts # 十维度解读引擎(~1000行)
```

## 快速开始

```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 生产构建
npm run build

# 启动生产服务
npm run start
```

启动后访问 `http://localhost:3000`
