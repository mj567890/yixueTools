# 嘉嘉易学工具箱 (YixueTools)

基于现代前端技术栈构建的中国传统易学在线工具集，提供八字排盘、万年历、梅花易数、阴盘奇门遁甲等功能，采用新中式视觉风格。所有计算均在本地完成，个人信息不上传服务器。

## 技术栈

- **框架**: Next.js 16 (App Router + Turbopack)
- **UI**: React 19 + TypeScript 5.9
- **样式**: Tailwind CSS 4
- **历法**: lunar-javascript 1.7.7

## 玉匣择吉模块

据《许真君玉匣记》编纂，以国学文化教育为核心的择吉知识系统，集成于万年历页面。

### 五大知识维度

- **诸神所在**: 六甲旬首定位，展示天/地/人间/地府诸神方位，附古籍原文与文化解读
- **六甲旬空**: 天干地支配对余数推导，空亡二支标记，解释旬空文化含义
- **神煞值日**: 70+ 神煞词典（吉神/凶煞），每条附出处与详细释义，支持展开查看
- **建除十二神**: 建除满平定执破危成收开闭完整系统，含宜忌事项与吉凶判定
- **黄道黑道**: 12值日天神分类（青龙/明堂/天刑/朱雀等），黄道六神/黑道六神对照
- **二十八宿**: 当日星宿值日，附星宿歌诀与吉凶说明
- **天赦日**: 春戊寅/夏甲午/秋戊申/冬甲子，特殊吉日识别
- **国学小课堂**: 每日关键词详解 + 互动问答测验（建除/黄道/旬空三类题库）

### 流派说明

择吉术存在多种流派（《协纪辨方书》《钦定协纪辨方书》《玉匣记》等），本工具以《玉匣记》为主要依据，lunar-javascript 库数据为辅助校验，仅供文化学习参考。

## 每日穿衣指南模块

基于五行生克理论的穿衣颜色推荐系统，支持通用模式与八字个人定制两种模式。

### 双模式架构

- **通用模式（无八字）**: 以颜色五行为"我"、流日天干五行为"环境"，分析哪种颜色能在当天天时中获益。分为得时色（环境生我）、和谐色（与环境同）、进取色（我克环境）、消耗色（我生环境）、不利色（环境克我）五档
- **个人定制模式（有八字）**: 输入出生年月日时及性别，排命主八字后判断日主强弱与喜用神/忌神，再综合大运（10年周期）、流年、流月、流日天干五行的多层影响，以加权评分算法对五种颜色排序，输出最佳色/次吉色/可选色/消耗色/忌穿色

### 评分算法

个人模式采用多因子加权评分：喜用神基础分(+4) / 忌神基础分(-4)，大运生助(+1.5)/克制(-1)，流年(+1/-0.5)，流月(+0.8/-0.4)，流日(+1.2/-0.8)，五行同气(+0.5)。

## 阴盘奇门遁甲模块

- **排盘核心**: 拆补法定局、地盘飞布（阳顺阴逆）、天盘九星/人盘八门/神盘八神转盘
- **隐干系统**: 伏吟/非伏吟双分支算法，地盘追踪法
- **月将隐支**: 中气定月将，月将安时支方位，顺时针排12地支
- **十二建除**: 时支起建，固定方位排列
- **九宫可视化**: 3x3封闭网格，外圈标注（月将+隐干+建星），空亡/马星标记
- **格局分析引擎**: 干落宫定位、旺相休囚、伏吟/反吟/击刑/入墓/门迫检测、综合吉凶判断

## 八字模块核心能力

- **排盘计算**: 四柱干支、十神、藏干、纳音、日主强弱、五行力量分析
- **大运系统**: 大运排列、起运年龄、五行评分、运势等级
- **流年系统**: 流年干支、合冲检测、评分、纳音、简洁断语
- **流年联动详解**: 原局联动(逐柱五行生克+冲合)、大运联动(五行互动+吉凶倾向)、四维断语(事业/财运/感情/健康)、关键提示(风险/机会)
- **十维度解读**: 性格/学业/事业/财运/婚姻/家庭/子女/健康/贵人/生命节律
- **五行雷达图**: SVG 五角雷达图，支持原局/+大运/+流年三层叠加对比

## 快速开始

```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 生产构建
npm run build

# 启动生产服务
npm run start
```

启动后访问 `http://localhost:3000`
