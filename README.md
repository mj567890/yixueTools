# 嘉嘉易学工具箱 (YixueTools)

基于现代前端技术栈构建的中国传统易学在线工具集，提供八字排盘、万年历、梅花易数、阴盘奇门遁甲等功能，采用新中式视觉风格。所有计算均在本地完成，个人信息不上传服务器。

## 技术栈

- **框架**: Next.js 16 (App Router + Turbopack)
- **UI**: React 19 + TypeScript 5.9
- **样式**: Tailwind CSS 4
- **历法**: lunar-javascript 1.7.7

## 阴盘奇门遁甲模块

- **排盘核心**: 拆补法定局、地盘飞布（阳顺阴逆）、天盘九星/人盘八门/神盘八神转盘
- **隐干系统**: 伏吟/非伏吟双分支算法，地盘追踪法
- **月将隐支**: 中气定月将，月将安时支方位，顺时针排12地支
- **十二建除**: 时支起建，固定方位排列
- **九宫可视化**: 3x3封闭网格，外圈标注（月将+隐干+建星），空亡/马星标记
- **格局分析引擎**: 干落宫定位、旺相休囚、伏吟/反吟/击刑/入墓/门迫检测、综合吉凶判断

## 八字模块核心能力

- **排盘计算**: 四柱干支、十神、藏干、纳音、日主强弱、五行力量分析
- **大运系统**: 大运排列、起运年龄、五行评分、运势等级
- **流年系统**: 流年干支、合冲检测、评分、纳音、简洁断语
- **流年联动详解**: 原局联动(逐柱五行生克+冲合)、大运联动(五行互动+吉凶倾向)、四维断语(事业/财运/感情/健康)、关键提示(风险/机会)
- **十维度解读**: 性格/学业/事业/财运/婚姻/家庭/子女/健康/贵人/生命节律
- **五行雷达图**: SVG 五角雷达图，支持原局/+大运/+流年三层叠加对比

## 快速开始

```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 生产构建
npm run build

# 启动生产服务
npm run start
```

启动后访问 `http://localhost:3000`
