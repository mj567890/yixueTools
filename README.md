# 嘉嘉易学工具箱 (YixueTools) v5.0

基于现代前端技术栈构建的中国传统易学在线工具集，提供八字排盘、万年历、梅花易数、六爻纳甲、奇门遁甲、起名测名等功能，采用新中式视觉风格。所有计算均在本地完成，个人信息不上传服务器。

> 嘉嘉易学工具箱，源自 2006 年最初编写的「嘉嘉易学万年历」，后更名为「嘉嘉易学工具箱」。时隔二十年，借助全新开发库对软件进行重制，完善当年未能完成的功能模块，致敬那段青涩而专注的青春。本软件完全免费。

## 玉匣择吉模块

据《许真君玉匣记》编纂，以国学文化教育为核心的择吉知识系统，集成于万年历页面。

### 五大知识维度

- **诸神所在**: 六甲旬首定位，展示天/地/人间/地府诸神方位，附古籍原文与文化解读
- **六甲旬空**: 天干地支配对余数推导，空亡二支标记，解释旬空文化含义
- **神煞值日**: 70+ 神煞词典（吉神/凶煞），每条附出处与详细释义，支持展开查看
- **建除十二神**: 建除满平定执破危成收开闭完整系统，含宜忌事项与吉凶判定
- **黄道黑道**: 12值日天神分类（青龙/明堂/天刑/朱雀等），黄道六神/黑道六神对照
- **二十八宿**: 当日星宿值日，附星宿歌诀与吉凶说明
- **天赦日**: 春戊寅/夏甲午/秋戊申/冬甲子，特殊吉日识别
- **国学小课堂**: 每日关键词详解 + 互动问答测验（建除/黄道/旬空三类题库）

### 流派说明

择吉术存在多种流派（《协纪辨方书》《钦定协纪辨方书》《玉匣记》等），本工具以《玉匣记》为主要依据，lunar-javascript 库数据为辅助校验，仅供文化学习参考。

## 每日穿衣指南模块

基于五行生克理论的穿衣颜色推荐系统，支持通用模式与八字个人定制两种模式。

### 双模式架构

- **通用模式（无八字）**: 以颜色五行为"我"、流日天干五行为"环境"，分析哪种颜色能在当天天时中获益。分为得时色（环境生我）、和谐色（与环境同）、进取色（我克环境）、消耗色（我生环境）、不利色（环境克我）五档
- **个人定制模式（有八字）**: 输入出生年月日时及性别，排命主八字后判断日主强弱与喜用神/忌神，再综合大运（10年周期）、流年、流月、流日天干五行的多层影响，以加权评分算法对五种颜色排序，输出最佳色/次吉色/可选色/消耗色/忌穿色

### 评分算法

个人模式采用多因子加权评分：喜用神基础分(+4) / 忌神基础分(-4)，大运生助(+1.5)/克制(-1)，流年(+1/-0.5)，流月(+0.8/-0.4)，流日(+1.2/-0.8)，五行同气(+0.5)。

## 阴盘奇门遁甲模块

- **排盘核心**: 拆补法定局、地盘飞布（阳顺阴逆）、天盘九星/人盘八门/神盘八神转盘
- **隐干系统**: 伏吟/非伏吟双分支算法，地盘追踪法
- **月将隐支**: 中气定月将，月将安时支方位，顺时针排12地支
- **十二建除**: 时支起建，固定方位排列
- **九宫可视化**: 3x3封闭网格，外圈标注（月将+隐干+建星），空亡/马星标记
- **格局分析引擎**: 干落宫定位、旺相休囚、伏吟/反吟/击刑/入墓/门迫检测、综合吉凶判断

## 六爻纳甲排盘模块

基于京房纳甲体系的完整六爻排盘与分析系统，依据《增删卜易》《卜筮正宗》标准实现。

### 四种起卦方式

- **时间起卦**: 基于农历年月日时干支推导上下卦与动爻
- **铜钱起卦**: 三枚铜钱摇六次，支持快捷选择与3D翻转动画
- **数字起卦**: 输入心中所想数字，结合时辰推导卦象
- **手动起卦**: 直接指定每爻的老阳/少阳/少阴/老阴

### 双流派支持

- **京房纳甲** (完整): 标准纳甲天干地支、八宫归属、世应定位
- **藏山卜** (Beta): 骨架已搭建，纳甲规则待后续填充

### 双模式切换

- **白话模式**: 通俗易懂的分析文案，适合初学者
- **专业模式**: 完整术语展示，含旺衰、飞伏关系、动爻变化详情

### 10步排盘管线

起卦 → 识卦(二进制O(1)查表) → 定宫(八宫归属) → 装纳甲(天干地支) → 排六亲 → 排六神 → 定世应 → 求伏神 → 算旺衰(月建日辰+旬空月破) → 组变卦

### 6层分析引擎

1. **定用神**: 根据10种场景(求财/问病/考试/婚恋/事业/失物/官司/出行/宅葬/杂占)确定用神六亲
2. **查原忌仇**: 推导原神、忌神、仇神
3. **评旺衰**: 月建旺衰 + 日辰生克冲合 + 旬空月破
4. **动爻分析**: 动爻对世爻的生克影响
5. **综合判断**: 加权评分(0-100)，输出吉/凶/平趋势
6. **文案输出**: 白话总结 + 专业总结 + 应期推测

### 核心数据

- 八宫64卦完整数据表(含世应位置、上下经卦、六爻阴阳)
- 京房纳甲天干地支对照表(8经卦 x 内外卦)
- 六神起法表(10天干 → 初爻六神)
- 六亲二维查表(宫五行 x 爻五行 → 六亲)
- 月建旺衰表(12月支 x 5五行 → 旺相休囚死)
- 旬空表(60甲子 → 空亡地支对)

## 起名测名模块

结合五格数理、三才配置、生辰八字的科学取名测名系统，纯前端计算。

### 双功能入口

- **测名分析**: 输入完整姓名，可选出生信息，输出五格数理、三才配置、八字匹配、音律字义、综合断语
- **起名推荐**: 输入姓氏和性别，可选出生信息，三阶段管线生成推荐名（笔画组合枚举 → 候选字筛选 → 多维评分排序）

### 五维评分体系

八字匹配(30%) + 五格数理(25%) + 三才配置(15%) + 音律分析(15%) + 字义分析(15%) = 综合评分(0-100)

### 核心引擎

- **五格计算**: 天格/人格/地格/总格/外格，支持单姓单名、单姓双名、复姓单名、复姓双名四种结构
- **三才分析**: 125种天人地五行组合查表，输出性格/事业/健康描述
- **八字匹配**: 调用八字排盘推导喜用神五行，计算姓名五行与喜用神的契合度
- **字义分析**: 平仄搭配、声母韵母避重、谐音检测（200+不雅词库）、字形笔画平衡
- **起名引擎**: 枚举吉利笔画组合(top100) → 按喜用神五行筛字 → 性别/避字过滤 → 全维度评分排序

### 辅助工具

- **汉字查询**: 输入汉字查五行/康熙笔画/拼音/部首/释义；按五行+笔画数反查汉字
- **姓名学小常识**: 知识库文章列表，涵盖五格/三才/八字等基础概念

### 数据

- 800+取名常用字库（含康熙笔画、五行、拼音、声调、部首、释义、性别倾向）
- 81数理吉凶表 + 125三才配置表 + 60+偏旁五行映射 + 80+复姓列表
- 康熙笔画字典兜底，覆盖非字库汉字

## 八字模块核心能力

- **排盘计算**: 四柱干支、十神、藏干、纳音、日主强弱、五行力量分析
- **大运系统**: 大运排列、起运年龄、五行评分、运势等级
- **流年系统**: 流年干支、合冲检测、评分、纳音、简洁断语
- **流年联动详解**: 原局联动(逐柱五行生克+冲合)、大运联动(五行互动+吉凶倾向)、四维断语(事业/财运/感情/健康)、关键提示(风险/机会)
- **十维度解读**: 性格/学业/事业/财运/婚姻/家庭/子女/健康/贵人/生命节律
- **五行雷达图**: SVG 五角雷达图，支持原局/+大运/+流年三层叠加对比

## 快速开始

```bash

npm run dev

# 生产构建
npm run build

# 启动生产服务
npm run start
```

启动后访问 `http://localhost:3000`
